# ProxHunt Discord Bot

Discord bot for the ProxHunt RFID platform.

## Installation

```bash
npm install
```

## Configuration

Create a `.env` file:

```env
DISCORD_BOT_TOKEN=your_bot_token
PROXHUNT_API=http://localhost:8080
DISCORD_CHANNEL_ID=your_channel_id
```

## Usage

```bash
npm start
```

## Commands

- `!leaderboard` / `!lb` - Show top collectors
- `!stats` - View discovery stats
- `!help` - Show help
